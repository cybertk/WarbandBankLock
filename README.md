# WarbandBankLock

> Prevents Warband Bank access conflicts across multiple online sub-accounts

WarbandBankLock is a lightweight addon that resolves simultaneous access issues for the Warband Bank when multiple sub-accounts under the same Battle.net account are logged in concurrently. It automatically restricts Warband Bank permissions to a single active account while blocking interactions from others.

WarbandBankLock adds a red X <img src=./Screenshots/screenshot-icon.png width=14/> to the Warband Bank spell icon for restricted accounts, making it clear when access is blocked. This prevents accidental use and removes the need for trial-and-error or memorization.

WarbandBankLock requires no manual configuration, and maintains minimal resource usage. Making it ideal for players playing multiple characters.
